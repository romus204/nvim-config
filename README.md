## Installation

1. Clone this repository:
    ```bash
    git clone https://github.com/romus204/nvim-config.git ~/.config/nvim
    ```

2. Install [Neovim](https://neovim.io/).

3. Launch Neovim and install plugins:
    ```bash
    nvim
    :Lazy sync
    ```
